Slide-ready equation summary
Algo 1
Modified MADDPG with TDec
If mi = 0:   ri(1) = -4.95 Di / Dmax
ri(2) = 0.05 R(Ci) - 0.5 log5(Pi) - Ai / 20
If mi = 1:   ri(1) = -4.95 Di / Dmax - 0.5 log5(Pi)
ri(2) = 0.05 R(Ci) - Ai / 20
rg = -(1 / N) Σi=1N (Ii + 60) / 60
Task-decoupled reward: task 1 stresses V2V completion and power, task 2 stresses V2I satisfaction and AoI freshness, with a shared interference penalty.
Algo 2
Modified MADDPG
ri = -4.95 Di / Dmax - Ai / 20 + 0.05 R(Ci) - 0.5 log5(Pi)
rg = -(1 / N) Σi=1N (Ii + 60) / 60
Merged local objective plus global interference suppression. This is the most direct multi-agent reward formulation in the repo.
Algo 3
MADDPG_FDec
ri = -4.95 Di / Dmax - Ai / 20 + 0.05 R(Ci) - 0.5 log5(Pi)
Same local reward form as Algo 2, but training does not use a separate global interference reward.
Algo 4
Centralized DDPG
ri = -4.95 Di / Dmax - Ai / 20 + 0.05 R(Ci) - 0.5 log5(Pi)
r = (1 / N) Σi=1N ri
One centralized DDPG agent controls all platoons jointly, so the learning signal is the average reward over platoons.

Takeaway

  • Algo 1 separates the objective into two local tasks and one global interference term.
  • Algo 2 collapses the local terms into a single per-agent reward, while keeping the global penalty.
  • Algo 3 keeps only the local reward, so learning becomes more decentralized.
  • Algo 4 uses the same platoon-level utility as Algo 2/3, but trains one centralized DDPG agent with the average reward.
  • Common structure: reward higher V2I threshold satisfaction, penalize stale AoI, residual V2V demand, and excessive transmit power.
