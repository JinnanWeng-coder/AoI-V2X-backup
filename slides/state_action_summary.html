State and Action Design of Algo 1-4

Simplified from the source code by grouping identical observation and action structures
Slide-ready summary

Shared Per-Platoon Action

Algo 1, Algo 2, and Algo 3 use the same 3-dimensional local action for each platoon. Algo 4 outputs the same action components, but for all platoons jointly through one centralized actor.

ai = [ RBi, Mi, Pi ]
RBi
resource block selection
Mi
mode selection: 0 = inter-platoon (V2I), 1 = intra-platoon (V2V)
Pi
transmit power selected by the platoon leader
Algo 4 form
one centralized vector: a = [a1, a2, ..., aN]

Shared Per-Platoon State

Algo 1, Algo 2, and Algo 3 all use the same local observation template. Algo 4 first builds the same local state for each platoon and then concatenates all platoon states into one global input.

si = [ V2Iabsi, V2Ifasti, V2Vabsi, V2Vfasti, Ii, AoIi, Li ]
V2Iabsi
large-scale V2I channel gain of platoon leader i
V2Ifasti
fast-fading V2I channel information over all RBs
V2Vabsi
large-scale V2V channels from leader to followers inside platoon i
V2Vfasti
fast-fading V2V channels to followers over all RBs
Ii
current interference level seen by platoon i
AoIi
normalized age of information of platoon i
Li
normalized remaining V2V load or demand
Algo 4 form
s = [s1, s2, ..., sN], i.e., a global flattened state

Quick Simplification

Algo 1 Algo 2 Algo 3 share the same local state and the same local action structure.

Algo 4 uses the same state components and same action components, but organizes them in a centralized global form.

Local MARL:   si → ai
Centralized DDPG:   [s1,...,sN] → [a1,...,aN]

Algorithm-by-Algorithm Summary

Algo State Action
Algo 1 Local per-platoon state si Local per-platoon action ai = [RBi, Mi, Pi]
Algo 2 Same as Algo 1: local state si Same as Algo 1: local action ai
Algo 3 Same as Algo 1: local state si Same as Algo 1: local action ai
Algo 4 Global flattened state s = [s1,...,sN] Centralized joint action a = [a1,...,aN]

So for slides, you can present Algo 1-3 together as decentralized per-platoon observation/action, and present Algo 4 as the centralized DDPG baseline.
